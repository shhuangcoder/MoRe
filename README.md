## Installation
1. Clone the repository from GitHub.

```shell
git clone https://github.com/shhuangcoder/MoRe.git
cd MoRe
```

2. Create conda environment.

```shell
conda create -n MoRe python=3.8
conda activate MoRe
```


3. Download the packages
```shell
pip install -r requirements.txt
```
